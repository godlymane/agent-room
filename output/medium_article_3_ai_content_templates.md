## The Template I Built (Full Structure)

### Category 1: EMAIL TEMPLATES (42 prompts)

**Cold Email Generator**
```
Subject: Write a cold email to [prospect_name] at [company] 
about [product_category]. Make it conversational, not salesy. 
Include one specific pain point from [industry]. 
End with a low-pressure call to action.
```
**→ Result:** Users copy/paste, send, get replies

**Sales Follow-Up**
```
The prospect didn't reply to email #1. Write a 2-sentence 
follow-up that's clever, doesn't sound desperate, and 
references something specific from the first email. 
Make them smile.
```

**Customer Breakup Email**
```
A customer is leaving. Write a 3-sentence exit email that:
1. Thanks them specifically
2. Asks why (makes them feel heard)
3. Leaves door open without sounding needy
```

→ **42 email templates total** = $5/person × 200 customers = $1,000/month

### Category 2: CONTENT CREATION (65 prompts)

**Viral Twitter Thread Generator**
```
Write a Twitter thread about [topic] that:
- Starts with a hook (question or surprising fact)
- Has 7-10 tweets
- Each tweet is punchy (under 280 chars)
- Includes 1 gif suggestion per tweet
- Ends with a call to action

**YouTube Title/Description Combo**
```

Target audience: [audience]
Generate 5 YouTube titles (under 70 chars) + 1 SEO-optimized 
description (300 chars) that includes hashtags & timestamps.

Write 3 versions:

Each 150-200 words.

→ **65 content templates** = $2/person × 400 customers = $800/month

### Category 3: CODE & AUTOMATION (85 prompts)

**Python Script Generator**
```
Write a Python script that [specific task]. 
Requirements:
- Works in Python 3.9+
- Uses only built-in or free libraries
- Includes error handling
- Has comments explaining each section
- Can be run via CLI with arguments
```

**Regex Pattern Builder**
```
I need a regex to match [pattern description]. 
Return ONLY the regex (no explanation). 
Test it against these examples: [examples]
```

**JavaScript DOM Manipulation**
```
I need to [DOM task] on a webpage. 
The HTML structure is: [structure]
Write vanilla JavaScript (no jQuery) that works in Chrome, 
Firefox, Safari. Include console.log for debugging.
```

→ **85 code templates** = $1/person × 300 customers = $300/month
